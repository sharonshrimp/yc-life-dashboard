# ⚡ Yi-Ching's Precision Hub (Project 60D)

一個專為 60 天健身挑戰設計的精準監控儀表板。

## 🛠 技術棧
- **Framework**: Next.js 15+ (App Router)
- **Database**: Firebase Firestore (Real-time Sync)
- **Styling**: Tailwind CSS
- **Visualization**: Chart.js
